## 1. Blogpost / Artikel

```
Prompt-Muster:
"Schreibe einen Blogpost ueber [THEMA].

 Zielgruppe: [z.B. Einsteiger / Fachleute / allgemein interessierte Leser]
 Tonalitaet: [locker / informativ / akademisch / motivierend]
 Laenge: [ca. X Woerter]
 Struktur: Einleitung -> 3-4 Abschnitte mit Unterueberschriften -> Fazit
 SEO-Fokus-Keyword: [optional]

 Besondere Anforderungen: [z.B. Praxisbeispiele, Quellenhinweise, CTA am Ende]"
```

**Ueberarbeitungs-Prompts:**
- "Kuerze den Text auf 60 %, behalte alle Kernaussagen."
- "Schreibe die Einleitung um — klarer Einstieg mit einer konkreten Frage."
- "Fuege 3 konkrete Beispiele in Abschnitt 2 ein."

---

## 2. Social-Media-Beitraege

```
Prompt-Muster:
"Erstelle 5 Social-Media-Beitraege fuer [LinkedIn / Instagram / X / Facebook]
 zum Thema: [THEMA]

 Tonalitaet: [professionell / locker / inspirierend]
 Mit passenden Hashtags: ja/nein
 Zeichenlimit: [Twitter: 280 / LinkedIn: ~1300 empfohlen]
 Call-to-Action: [optional, z.B. 'Link in Bio', 'Kommentiert eure Meinung']"
```

**Varianten:**
- Ankuendigung neuer Inhalte
- Zitat / Thought Leadership
- Behind-the-scenes
- Umfrage / Engagement-Post

---

## 3. Newsletter

```
Prompt-Muster:
"Schreibe einen Newsletter fuer [VERTEILER/THEMA].

 Betreff: [z.B. catchy, max. 60 Zeichen, kein Clickbait]
 Pre-Header: [max. 100 Zeichen, ergaenzt den Betreff]

 Struktur:
 - Kurze persoenliche Begruessung
 - Hauptthema: [BESCHREIBUNG, ca. 200-300 Woerter]
 - Kurz-Nachrichten: 2-3 Bulletpoints mit je 1-2 Saetzen
 - Abschluss + CTA (z.B. Terminhinweis, Link, Frage)

 Tonalitaet: [persoenlich-direkt / sachlich / motivierend]"
```

---

## 4. Marketing-Copy

```
Prompt-Muster fuer Produktbeschreibung:
"Schreibe eine ueberzeugende Produktbeschreibung fuer [PRODUKT/SERVICE].

 Zielgruppe: [BESCHREIBUNG]
 Hauptvorteile: [3-5 Bulletpoints]
 Unique Selling Point: [was unterscheidet es?]
 Tonalitaet: [direkt / emotional / technisch]
 Laenge: [kurz ~100 Woerter / mittel ~250 Woerter / lang ~500 Woerter]
 Format: [Fliesstext / Bulletpoints / beides]"

Prompt-Muster fuer Headline-Varianten:
"Schreibe 10 Headline-Varianten fuer [PRODUKT/KAMPAGNE].
 Zielgruppe: [BESCHREIBUNG]
 Stil-Mix: 3 emotional, 3 nutzenorientiert, 2 Frage-Format, 2 Dringlichkeit"
```

---

## 5. Formelle E-Mail / Brief

```
Prompt-Muster:
"Schreibe eine formelle E-Mail.

 Von: [Absender-Rolle, z.B. 'Projektleitung']
 An: [Empfaenger-Rolle, z.B. 'Kooperationspartner']
 Betreff: [BETREFF]
 Anlass: [BESCHREIBUNG]
 Gewuenschtes Ergebnis: [z.B. Terminvereinbarung, Rueckmeldung, Genehmigung]
 Tonalitaet: professionell, freundlich
 Laenge: knapp, max. 200 Woerter"
```

---

## 6. Bericht / Zusammenfassung

```
Prompt-Muster:
"Fasse [DOKUMENT/TEXTE] zusammen.

 Zielformat: [Executive Summary / Bericht / Stichpunkte]
 Laenge: [1 Seite / 3 Seiten / X Woerter]
 Struktur: Kontext -> Kernaussagen -> Schlussfolgerungen -> Empfehlungen
 Zielgruppe: [z.B. Fuehrungsebene / technisches Team]"
```

---

## Qualitaetspruefung

Nach der Generierung pruefen:
- **Fakten:** KI erfindet Details. Eigennamen, Zahlen, Zitate verifizieren.
- **Tonalitaet:** Zum Kanal und zur Marke passend?
- **Laenge:** Passt zum Medium und zur Aufmerksamkeit der Zielgruppe?
- **CTA:** Ist der naechste Schritt klar?

## Stiloptimierung (DeepL Write) — optional

Fuer hochwertige Stiloptimierung innerhalb einer Sprache:
```
1. deepl.com (kostenlos bis 500.000 Zeichen/Monat) oder DeepL Pro
2. Text einfuegen -> Zielsprache waehlen
3. "DeepL Write" fuer stilistische Optimierung innerhalb einer Sprache
```

Fuer hochwertige Uebersetzungen (z.B. DE -> EN):
```
1. deepl.com: Text einfuegen -> Zielsprache waehlen
2. Ergebnis ggf. mit DeepL Write finalisieren
```
