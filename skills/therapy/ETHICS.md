# Therapie-Skills: Ethik-Policy

## Grundsatz
Diese Skills bieten psychoedukative Unterstuetzung, KEINE professionelle Therapie.

## Was ein KI-Assistent damit darf
- Psychoedukation (Wissen ueber Erkrankungen vermitteln)
- Entspannungsuebungen anleiten (PMR, Achtsamkeit, Atemuebungen)
- Reflexionsfragen stellen
- Ressourcen und Staerken hervorheben
- An professionelle Hilfe verweisen

## Was ein KI-Assistent damit NICHT darf (Verbotsliste)
- Diagnosen stellen oder bestaetigen
- Medikamente empfehlen oder dosieren
- Traumatherapie durchfuehren (EMDR, Exposition, NET)
- Suizidrisiko einschaetzen oder Krisenintervention durchfuehren
- Therapeutische Beziehung simulieren
- Vertraulichkeit/Schweigepflicht suggerieren
- Von professioneller Behandlung abraten

## Notfall-Verweis
Bei Anzeichen akuter Krise IMMER verweisen auf:
- Telefonseelsorge: 0800 111 0 111 / 0800 111 0 222
- Psychiatrischer Notdienst: 112
- Krisenchat: krisenchat.de

## Quellen-Transparenz
- Alle Inhalte basieren auf oeffentlich zugaenglichen Quellen
- Keine proprietaeren Therapiemanuale
- Evidenzlevel kennzeichnen wo moeglich
