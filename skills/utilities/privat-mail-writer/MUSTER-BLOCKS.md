# Privat-Mail-Writer - Musterblöcke

Diese Blöcke sind kurze Bausteine. Beim Entwurf nur wenige passende Blöcke kombinieren und an das Kontaktprofil anpassen.

## Dank

```text
vielen Dank für Ihre Nachricht.
```

```text
danke für deine Nachricht.
```

```text
vielen Dank für die Einladung.
```

## Offizielle Absage

```text
leider muss ich den Termin am [Datum] absagen.
```

```text
leider kann ich den Termin am [Datum] nicht wahrnehmen.
```

```text
ich bitte um Verständnis, dass ich den Termin absagen muss.
```

## Ersatztermin

```text
Einen Ersatztermin können wir gerne abstimmen.
```

```text
Gerne melde ich mich wegen eines neuen Termins.
```

```text
Wenn möglich, würde ich mich über einen Alternativtermin freuen.
```

## Neutrale Begründung

```text
Aus organisatorischen Gründen ist mir die Teilnahme leider nicht möglich.
```

```text
Leider ist mir kurzfristig etwas dazwischengekommen.
```

```text
Aus persönlichen Gründen kann ich den Termin leider nicht wahrnehmen.
```

## Abschluss

```text
Vielen Dank für Ihr Verständnis.
```

```text
Danke für dein Verständnis.
```

```text
Ich wünsche Ihnen einen guten Tag.
```

## Kürzungsregel

Wenn der User "kurz" sagt, höchstens einen Dank, eine Kernbotschaft und einen Abschluss verwenden. Keine doppelten Entschuldigungen und keine ausführliche Begründung.
